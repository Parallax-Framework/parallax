ax.util:DrawPanelBlur (panel, passes, density, alpha, throttle, clip)
Draws a blur behind a panel (client-only).

Parameters:

  • panel Panel The panel to blur behind
  • passes number Number of blur passes (default 3)
  • density number Blur density multiplier
  • alpha number Alpha of blur overlay (0-255)
  • throttle number Minimum seconds between updates (optional)
  • clip boolean Whether to clip the blur to panel bounds

Usage:

    ax.util:DrawPanelBlur(myPanel, 3, 1.0, 180)
